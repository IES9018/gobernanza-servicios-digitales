# Glosario — Gobernanza de Servicios Digitales IES 9-018

---

| Término | Significado |
|---------|-------------|
| **Admin técnico** | Persona responsable de operar y mantener el servidor |
| **API** | Interfaz que permite que dos programas se comuniquen entre sí |
| **API key** | Clave secreta que identifica a quien usa una API |
| **Backup** | Copia de seguridad de los datos |
| **Bindear** | Asociar un puerto del servidor a un puerto del contenedor |
| **Contenedor** | Entorno aislado donde corre una aplicación (Docker) |
| **Docker** | Programa que permite crear y ejecutar contenedores |
| **DNS** | Sistema que traduce nombres de dominio a direcciones IP |
| **Hardcodeado** | Datos escritos fijos en el código fuente (ej: contraseñas) |
| **HTTPS** | Conexión cifrada entre el navegador y el servidor |
| **Log** | Registro de eventos que ocurren en el sistema |
| **Puerto** | Número que identifica un servicio en el servidor |
| **Proxy** | Servidor intermediario que gestiona el tráfico entre el usuario y la app |
| **SLA** | Acuerdo de nivel de servicio (Service Level Agreement) |
| **SSH** | Protocolo para acceder de forma segura a un servidor remoto |
| **Tailscale** | Red privada que permite acceder al servidor desde cualquier lugar |
| **Tunnel** | Conexión segura que expone un servicio local a internet |
| **URL** | Dirección web (ej: https://ejemplo.ies9018malargue.edu.ar) |
